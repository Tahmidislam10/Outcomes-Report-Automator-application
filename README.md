
# Outcomes Report Automator

The **Outcomes Report Automator** is a Python-based application designed to streamline and automate the process of generating academic outcomes reports. It removes manual effort by taking structured Excel input, validating the data, and automatically producing finalised PDF reports. The tool ensures accuracy, consistency, and efficiency for staff who regularly handle student performance documentation.
